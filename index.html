"Consulta fácil, rápida e especializada."

Sinopse

O projeto Cerana é um sistema multiplataforma para consulta através de perguntas ou pesquisa numa base de dados organizada em um sistema baseado no crowdsourcing. O intuito é construir o funcionamento de uma rede social que tenha a praticidade que o facebook traz e agregar a isso um sistema que funcione como o stackoverflow, voltado para a área ambiental/biologia e afins.
